Instruções e Ajuda

Esta é a tela de ajuda, para ler as instruções basta seguir os itens abaixo:

Como é a Navegação na Tabela no Computador ?

Para navegar entre os elementos da tabela periódica você deve usar o teclado do seu computador. As teclas para navegação são: tecla W para cima, tecla Z para baixo, tecla A para esquerda, a tecla D para direita e a tecla Q é um atalho para encontrar um elemento pela sua localização na tabela, linha e coluna.

Como é a Navegação na Tabela no Celular ?

Para navegar entre os elementos da tabela periódica usando o seu smartphone, você deve usar dois dedos fazendo movimentos na tela, para direita, esquerda, cima e para baixo. Ou seja, caso a direção seja baixo basta deslizar os dedos para este mesmo sentido. Caso para cima, deslizar os dedos nesta mesma direção. Caso para o lado direito basta deslizar os dedos para esta mesma direção, assim como para o lado esquerdo deslizar os dedos para esta mesma direção.

Existe Alguma tecla de Atalho para Navegação em Computadores ?

Sim. A tecla de atalho é a tecla Q . Com esta tecla você poderá buscar um elemento especifico na tabela periódica, digitando a linha e coluna do mesmo.

Quais informações vou encontrar na tabela periódica web ?

Durante a navegação na tabela os campos exibidos na tela apresentará as seguintes informações: primeiro o número atômico, segundo a letra ou símbolo que representa o elemento, em seguida o nome do elemento e por fim a massa atômica, que é uma sequência numérica.

Qual sintetizador de voz devo usar ?

Recomendamos usar o software sintetizador de voz de sua preferência. Contudo há o NVDA um software sintetizador de voz gratruito, assim como outros disponíveis na internet, para sistemas operacionais como Windowns. Para navegação em celulares estilo smartphone, usa-se o assistente de acessibilidade do dispositivo. Você pode habilitar o sintetizador de voz, indo até as configurações do seu aparelho.

Como devo usar a tabela periódica web ?

A tabela periódica web acessível pode ser explorada de diversas formas. Ela foi desenvolvida com o objetivo principal de disponibilizar informações básicas sobre a tabela periódica e que poderão contribuir para um melhor entendimento e inclusão das pessoas com deficiência visual no ambiente escolar, auxíliando em alguma atividade que envolva seu uso, mas ela também pode ser usada em qualquer lugar, desde que se tenha acesso a um computador e/ou celular com conexão a internet.
